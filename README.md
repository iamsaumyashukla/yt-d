# YouTube Downloader

A simple Node.js web application to download YouTube videos.

## Local Setup
1. Make sure Node.js is installed on your computer.
2. Run `npm install` to install the required dependencies.
3. Run `npm start` to start the server.
4. Open `http://localhost:4000` in your web browser.

## Deployment to Render / Railway
This app is pre-configured to be deployed easily.
1. Upload this entire repository to GitHub.
2. Create a new Web Service on [Render](https://render.com/) or Railway.
3. Connect your GitHub repository.
4. The host will automatically run `npm install` and `npm start`.
